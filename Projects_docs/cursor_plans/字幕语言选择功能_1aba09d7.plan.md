# Hard-burned 字幕语言选择

## 现状分析

- WhisperKit 0.9.0 的 `DecodingOptions` 已支持 `language: String?` 参数（[Configurations.swift](../../Library/Developer/Xcode/DerivedData) 第 121 行）
- 当前代码未设置该参数，Whisper 默认自动检测语言
- 应用的 15 种界面语言定义在 [LanguageManager.swift](Shared/Sources/MyAppCore/Localization/LanguageManager.swift) 中
- zh-Hans 和 zh-Hant 在 Whisper 中都映射为 "zh"，因此实际 Whisper 语言为 14 种

## 语言代码映射

应用语言代码到 Whisper 语言代码的映射：

- `en` -> `en`, `zh-Hans` -> `zh`, `zh-Hant` -> `zh`, `hi` -> `hi`
- `es` -> `es`, `fr` -> `fr`, `ar` -> `ar`, `bn` -> `bn`
- `ru` -> `ru`, `pt` -> `pt`, `ur` -> `ur`, `ja` -> `ja`
- `ko` -> `ko`, `vi` -> `vi`, `id` -> `id`

## 实现方案

### 1. 新增字幕语言枚举/映射

在 [SRTFormatter.swift](Projects/SilenceCut_App/ios_workspace/SilenceCut/Sources/SilenceCut/Captions/SRTFormatter.swift) 中添加 `SubtitleLanguage` 枚举或在现有代码中添加映射工具：

```swift
public enum SubtitleLanguage: String, CaseIterable, Codable {
    case en, zhHans, zhHant, hi, es, fr, ar, bn, ru, pt, ur, ja, ko, vi, id
    
    public var whisperCode: String { ... }  // 映射到 Whisper 语言代码
    public var displayName: String { ... }  // 复用 LanguageManager.displayNames
    public static let `default`: SubtitleLanguage = .en
}
```

### 2. 修改 FillerWordDetectionService 协议和实现

**协议层** - [FillerWordDetectionService.swift](Projects/SilenceCut_App/ios_workspace/SilenceCut/Sources/SilenceCut/Services/FillerWordDetectionService.swift)：
- `transcribe(asset:)` 改为 `transcribe(asset:language:)` 或新增带语言参数的重载
- 保持默认值 `language: String? = nil`（向后兼容）

**实现层** - 同文件 `WhisperFillerWordDetectionService`：
- 在 `detectFillerWords` 中接收语言参数
- 设置 `options.language = language`（约第 237-241 行区域）

### 3. ExportViewModel 传递语言参数

[ExportViewModel.swift](Projects/SilenceCut_App/ios_workspace/SilenceCut/Sources/SilenceCut/ViewModels/ExportViewModel.swift)：
- `init` 新增 `subtitleLanguage` 参数（默认 `.en`）
- 在 `startExport()` 中调用 `service.transcribe(asset:language:)` 时传入

### 4. EditorView 添加语言选择 UI

[EditorView.swift](Projects/SilenceCut_App/ios_workspace/SilenceCut/Sources/SilenceCut/Views/EditorView.swift)：
- 新增 `@State private var selectedSubtitleLanguage: SubtitleLanguage = .en`
- 在 subtitle picker（约第 345-358 行）下方，当 `selectedCaptionMode == .hardBurned` 时显示语言 Picker
- 创建 `ExportViewModel` 时传入 `subtitleLanguage`

### 5. 本地化

为新增的 UI 文案（如 "Subtitle Language"）在 14 种语言的 `Localizable.strings` 中添加翻译。

## 数据流

```
EditorView (selectedSubtitleLanguage)
  -> ExportViewModel.init(subtitleLanguage:)
    -> startExport()
      -> service.transcribe(asset:language:)
        -> detectFillerWords(in:language:)
          -> DecodingOptions.language = whisperCode
            -> whisperKit.transcribe(audioPath:decodeOptions:)
```

## 注意事项

- Whisper 的 `language` 参数接受 ISO 639-1 代码（如 "en", "zh", "ja"）
- 当 `language = nil` 时 Whisper 自动检测语言（当前行为），设置后强制使用指定语言
- 需确认小型 Whisper 模型（tiny/base）对非英语语言的识别质量可能较低，可在 UI 中提示
- Mock 服务 ([MockServices.swift](Projects/SilenceCut_App/ios_workspace/SilenceCut/Tests/SilenceCutTests/Services/MockServices.swift)) 也需要同步更新协议
