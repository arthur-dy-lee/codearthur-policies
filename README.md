## URL 规则

所有文档支持通过 URL 参数动态注入 App 名称：

```
https://arthur-dy-lee.github.io/codearthur-policies/en/privacy.html?app=FireApp
https://arthur-dy-lee.github.io/codearthur-policies/en/terms.html?name=FireApp
```

页面内的 JS 会读取 `app` 或 `name` 参数，自动替换文档中的 App 名称占位符。一套文档服务矩阵中的所有 App。

## App 中的调用方式

在 iOS App 中使用 `SFSafariViewController` 或 `WKWebView` 打开：

```swift
let appName = "YourAppName"
let privacyURL = "https://arthur-dy-lee.github.io/codearthur-policies/en/privacy.html?name=\(appName)"
let termsURL = "https://arthur-dy-lee.github.io/codearthur-policies/en/terms.html?name=\(appName)"
```

提交 App 审核时，在 App Store Connect 中填入 Privacy Policy URL。

## 启用 GitHub Pages

1. 进入仓库 **Settings > Pages**。
2. **Source** 选择 `Deploy from a branch`。
3. **Branch** 选择 `main`，文件夹选 `/ (root)`。
4. 保存，稍等片刻即可通过上述地址访问。
